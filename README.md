# 𐕣 METAL FORGE 𐕣

> **Status:** Iron Vault Connected • Production Live via Vercel
> **Target Audio Engines:** Suno AI, Udio AI, and Multimodal Visual Ingestion

Welcome to the **Metal Forge**, a feature-rich, high-contrast prompt-crafting studio designed specifically to forge hyper-optimized style modifiers, lyrical themes, and visual parameters for AI music engines. 

---

## ⚡ Core Features

* **Atmosphere Theme Matrix:** Dynamic color-shifting engine toggles seamlessly across *Hellfire Red*, *Toxic Sludge Green*, and *Frozen Fjord Blue* environments.
* **Style Modifiers Matrix:** A collapsible array of raw musical components (blast beats, HM-2 distortion, ripping solos, etc.) to stack sonic blueprints instantly.
* **Targeted Component Locking:** Freeze individual generated outputs (like killer lyrics or generated titles) while re-rolling the remaining project parameters.
* **Instant Purge Workspace:** A dedicated cleanup pipeline (`Reset Forge`) designed to instantly sweep the active workspace and inputs clean for a fresh project slate.

---

## 🛠️ Architecture & Tech Stack

This suite is engineered as a lightweight, lightning-fast application designed to minimize build bloat and execution stutter:

* **Frontend:** Pure Semantic HTML5 & Modern JavaScript (ES6+)
* **Styling:** Dynamic utility-first CSS via the Tailwind CDN
* **Backend Interface:** Serverless API Edge Function (`api/forge.js`) securely routing prompt packages straight to the Gemini Pro API model.
* **Deployment Hosting:** Automated production pipeline synced via Vercel GitHub webhooks.

---

## 𐕣 How to Wield the Forge

1.  **Configure Parameters:** Input a project song title, band influence, vocal styles, or upload a reference image into the *Visceral Vision* module.
2.  **Ignite Generation:** Slam the primary **FORGE THE TRACK** command button.
3.  **Refine & Lock:** Review outputs. Toggle the safety locks on fields you want to preserve for the next prompt compilation.
4.  **Sweep Clean:** Hit **RESET FORGE** to instantly purge your workspace containers and return to a fresh onboarding slate.
